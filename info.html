О проекте

Цель и задачи

Цель курсовой работы — разработка интерактивного веб-сайта, посвящённого визуализации исторических образов Древней Руси с использованием современных веб-технологий.

Основные задачи:

  • Изучить принципы работы с 3D-графикой в браузере (библиотека Three.js)
  • Реализовать адаптивный при помощи современного CSS3
  • Научиться проводить базовую вёрстку веб-страниц
  • Обеспечить кросс-платформенную совместимость (ПК, планшеты, смартфоны)

Используемые технологии

Проект реализован на «чистых» веб-технологиях без фреймворков:

  • HTML5 — семантическая разметка
  • CSS3 — адаптивная верстка, flexbox, grid, переменные, backdrop-filter
  • JavaScript (ES6+) — интерактивность, управление DOM
  • Three.js — отображение и анимация 3D-модели картины
  • GLTF/GLB — формат 3D-модели

Описание функционала

Сайт состоит из трёх основных разделов:

  • Главная страница — содержит 3D-модель картины в рамке, которая слегка поворачивается за курсором, а также информационный блок с историческим контекстом.
  • Галерея — интерактивная коллекция работ художников XIX века с подписями и возможностью просмотра в увеличенном виде.
  • Информация — техническое описание проекта и его реализации.

Все страницы адаптированы под различные устройства.

Заключение

В ходе выполнения курсовой работы были освоены ключевые аспекты современной веб-разработки: работа с 3D-графикой, создание адаптивных интерфейсов, применение современных CSS-эффектов. Проект демонстрирует синтез исторической темы и цифровых технологий, что делает его не только технически, но и содержательно значимым.
